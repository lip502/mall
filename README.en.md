# 基于springboot手机商城管理系统

#### Description
前台：商品查看（分类查看，商品详情，），支付管理 个人中心（增删改查）
后台：用户管理 分类管理（一级分类和二级分类） 订单管理 商品管理 （增删改查）


Spring boot+mysql手机商城（带后台管理系统）,没有加微信或支付宝支付功能，可以基于这个项目做二次开发
运行环境
jdk8+mysql5.6+IntelliJ IDEA+maven
